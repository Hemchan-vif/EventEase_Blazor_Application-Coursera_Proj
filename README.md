# EventEase Application (Coursera Demo Project)

This repository contains the EventEase Blazor application developed as part of a Coursera course assignment.

## Project Overview
EventEase is a simple demo application designed to demonstrate:
- Blazor component development
- Two-way data binding
- Routing and navigation
- Input validation
- Basic state management
- Attendance tracking

## Copilot Usage
Microsoft Copilot was used to generate initial components, assist with routing setup, debug data binding issues, and optimize validation logic throughout development.

## Note
This project is intended for educational purposes only.
